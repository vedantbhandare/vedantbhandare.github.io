My Projects

Bio-monitoring and Response System

The BioFeedback System is an intelligent wellness platform that turns your body’s real-time signals—like stress levels, recovery quality, and cognitive load—into clear, personalized guidance. Combined with lightweight sensors and adaptive AI, it acts as a real-time health coach.

3D Object Gesture Viewer

The 3D Object Gesture Viewer is an interactive system that allows users to rotate, scale, and manipulate 3D objects using hand gestures. I built the complete gesture-recognition pipeline using OpenCV and MediaPipe to detect hand landmarks, extract features, and classify gestures in real time. The detected gestures are mapped to object transformations inside a 3D rendering engine, creating a smooth, intuitive, touch-free interaction experience.

Smart Obstacle and Height Detection

Engineered a wearable assistive device for the visually impaired. Programmed an Arduino Uno R3 in C++ to process real-time data from HC-SR04 ultrasonic sensors. Achieved a reliable detection range of 2 meters with 95% accuracy using haptic feedback.

IR Obstacle Detector

Constructed and calibrated a compact infrared proximity sensor using an IR LED emitter, photodiode receiver, and LM358 op-amp. Calibrated for consistent detection thresholds for object avoidance applications.

Footstep Electric Generator

Designed a renewable energy proof-of-concept capturing kinetic energy. Integrated piezoelectric sensors into footwear to convert mechanical stress into electrical energy, successfully generating output for low-energy devices.
